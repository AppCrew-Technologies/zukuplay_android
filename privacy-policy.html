7. GOOGLE USER DATA (ZUKUPLAY - iOS/ANDROID)

In order for the application to be able to play, download and upload video, audio files on Google Drive, you need to grant the application access to Google Drive files. Application access to files is limited to certain audio type file extensions (mp3, mp4, flac, etc.).

When you connect Google Drive within the application, we will use the Google Drive API and access some user data from your Google profile and/or Google Account Identifier ("account Id").

"Zukuplay" use and transfer of information received from Google APIs to any other app will adhere to Google API Services User Data Policy, including the Limited Use requirements.

1) Accessing Google User Data

Zukuplay accesses Google resources from the device where you use Zukuplay. Your device communicates directly with the Google APIs.

Zukuplay sends users to https://accounts.google.com/o/oauth2/auth for authorization and does not directly access or collect user data used by Google Auth.

2) Using Google User Data

Zukuplay includes functions that you can execute in order to read or modify your own data (or data shared with you). This can only happen after you provide a token, which requires that you authenticate yourself as a specific Google user and authorize these actions.

3) Collecting Google User Data

userinfo.email (view your email address), userinfo.profile (see your personal info. including any personal info you've made publicly available), and openid (authenticate using OpenID Connect) are required to be used by Google Auth.

Zukuplay allows you to manage your Google Drive files and therefore the default scopes also include drive (read/write access to your Google Drive) and drive.appdata (manage configuration folder in your Google Drive).

However, Zukuplay never receives your data or the permission to access your data in your Google Drive account. The owners of the project can only see anonymous, aggregated information about usage of tokens obtained through its OAuth client, such as which APIs and endpoints are being used.

4) Sharing Google User Data

Zukuplay only communicates with Google APIs. No user data is shared with the owners of Zukuplay, or any other party.

Notice: If you experience a crash while using Zukuplay, the crash report sent to our servers may contain some information about the operating system you are using.
